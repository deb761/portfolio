Overall Formatting

The overall site colors are a dusky purple border with a tan background. The default text color is black. The first three header levels are a darker version of the border.

The general font-family for the site is Arial with backup fonts Helvetica and Sans Serif. The general text size is 1.2em.

Site Header / Branding

The header area font is inherited from the body.

Navigation Links

Site navigation uses the font-family Arial, and is inherited from the body. The colors are modified to be various shades of magenta as the user hoovers over, follows, and returns from a link. The nav section takes up 20% of the total page width and is aligned to the right.

Content Section

The webpage content section uses padding and margin of 2.5% for the top, left and right, and 2% for the bottom. The section is aligned to the left, and is assigned 70% of the total page width. The left column is set to end at the conclusion of this section so that the footer does not move into the nav section.

Site Footer

The footer has a background color of light tan inherited from the body. Each footer has a horizontal rule in a color similar to the border, and a link to my email address.
