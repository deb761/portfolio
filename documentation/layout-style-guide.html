Overall Layout

The overall layout of the site has a header section with the page title and an unordered list of links to my social sites. This is followed by a navigation section and finally by the content section and footer.

The layout of these regions is sequential for screen sizes less than 480 pixels. After this, the social sites list gets moved to the top right of the header section, and the navigation section gets moved to float on the right. The content section width is adjusted so that it will not overlap the navigation section. This content section width is defined as a percentage of the body width, and this percentage is adjusted to avoid a large gap between the content and navigation sections as the page width changes.
