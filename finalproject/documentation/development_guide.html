Site Directory Structure

This site combines information and coordination.

Root Directory
The root directory contains the following files:
index.html
Home page for the site, explains the site and club purpose.
about.html
More information about the club and its members.
history.html
History of the club.
news.html
Current news in a blog style, with most recent news first.
meetings.html
Calendar of meetings.
membership.html
Explains membership requirements and provides a form to apply for membership.
library.html
Explains purpose of library and provides a means to reserve materials.
images
Images shown on the web pages, other than photographs uploaded by members.
styles
Style sheet files for the site.
data
Club database.
