Overall Document Styles

html {
    background-color: #77aaee; /* light blue */
}

body {
    width: 90%;
    margin-right: auto;
    margin-left: auto;
    padding: 2.5% 2.5% 2%;
    background: #eeddcc; /* light tan */
    font-family: Arial, Helvetica, sans-serif;
    color: #253034; /* nearly black with a hint of green */
    font-size: 1.2em;
}

The html style sets the background color to a dusky purple, and the body style sets the background color to tan, with a width of 90% of the page, so it has the effect of a purple border around a beige document.

The body style also defines the base font for the pages and sets the size to a bit larger than the default.

Content Document Style

section#content {
    float: left;
    width: 70%;
    margin: 1em;
}

section#content:before,
section#content:after {
    content: " ";
    display: table;
}

section#content:after {
    clear: both;
}

The content style sets the content section to 70% of the page width, and instructs the browser to float the section to the left of the page.

Nav Document Style

nav {
    float: right;
    margin: 1em;
    width: 20%;
    background-color: #A1B9D1;
    border: 1px solid #4F547C;
}

The nav style instructs the browser to float the section to the right of the page, changes the background color, and adds a border.

See-Through Class Style

.see-through {
    opacity: 0.8;
    float: left;
    margin: 0em 2em 2em 0em;
}

The see-through class style makes the image a bit sheer, puts a border around it, and floats it so that text flows around it to the right.

Border Class Style

.border {
    border-style: ridge;
    border-width: thick;
    border-color: blue;
}

The border class style puts a blue ridged border around the element.

Footer Document Style

footer {
    clear: both;
    font-size: 0.9em;
}

The footer style clears any left or right float formats that moy have been set so that the footer appears at the bottom of the page when main and aside are used.

Table Style

table, th, td {
    border-style: groove;
    border-color: #58749B;
}

th, td {
    padding: 15px;
}

caption {
    color: #445544;
    font-weight: bold;
}

The table element style specifies a border around any tables so that they are easily identified as tables.

OL Biblio Class Style

ol.biblio {
    list-style: none;
}

ol.biblio li {
    padding-bottom: .8em;
    margin-left: 2em;
    text-indent: -2em;
}

The biblio class sets up a hanging indent for the entries and adds some padding between them.
