Style Guide

Overall Layout

The overall layout of the pages has a header section followed by a member sign-in, then a site menu, followed by the page content. That is how the pages are ordered without CSS formatting and appear on small screens.

As the page gets wider, the login is moved to the top right corner, and the content is moved to the right of the site menu, as seen in the layout sketch. The floats are controlled with the clear-fix class.

.clearfix:before,
.clearfix:after {
    content: " ";
    display: table;
}

.clearfix:after {
    clear: both;
}

Overall Styles

html {
    background-color: #C32A2A;
}

body {
    background-color: #EFEFEF;
    width: 90%;
    margin-right: auto;
    margin-left: auto;
    padding: 2.5% 2.5% 2%;
    font-family: tahoma, arial, Helvetica, sans-serif;
    color: black;
    font-size: 1.2em;
}

The html style sets the background color to a deep red, and the body style sets the background color to light gray, with a width of 90% of the page, so it has the effect of a red border around a light gray document.

The body style also defines the base font for the pages to tahoma and sets the size to 1.2 times larger than the default.

.member {
    visibility: hidden;
}

The member class is used to control visibility of elements that are accessed by members only.

h1, h2, h3 {
    font-family: "Times New Roman", Georgia, serif;
    letter-spacing: .1em;
    color: #003763; /* dark navy */
    text-shadow: 1px 1px black;

The header font is set to a dark navy and a serif font.

Letterhead

#letterhead {
	display: inline-block;
}

#letterhead h1,h2,h3 {
	color: #C32A2A;
	letter-spacing: 0;
	margin: 0;
	padding: 0;
}

#letterhead h2 {
	font-size: .9em;
}

The letterhead section contains the club logo and name. The font color for the h1, h2, and h3 elements is changed to match the main color of the logo, and positioned relative to the header, as seen in the CSS from the section for displays wider than 650 pixels.

header {
    position: relative;
}

#letterhead {
    position: absolute;
    top: 60px;
    left: 255px;
}

Login

#login {
	font-size: .7em;
	border: solid thin;
	background-color: #B6B6B6;
	width: 16em;
}

#login h2 {
	font-family: Gotham, "Helvetica Neue", Helvetica, Arial, sans-serif;
	font-size: 1em;
	text-shadow: none;
	text-align: center;
}

#login label {
	cursor: pointer;
	display: inline-block;
	width: 65px;
	text-align: right;
}

#login input {
	font-size: inherit;
	width: 10em;
	padding-right: 2px;
}

The login uses a Gotham font that is 0.7em smaller than the body text. The login element is a form, and its labels and inputs are set to a fixed width that is just large enough for the text within. CSS does not allow using relative units for labels. The labels are also right-aligned.

Navigation

#mainNav {
    margin-right: 1em;
    width: 200px;
    background-color: #E8E4EB;
    border: 1px solid #4F547C;
}

#mainNav ol, #social ul {
	list-style-type: none;
	margin: 0.5em 1em;
	padding: 0;
}

/* Shift the image over a tad so it fits nicely in the nav bar */
#mainNav img {
	position: relative;
	left: -1em;
}

The Main Navigation section contains an ordered list of links to the other pages on the site. The list-style-type is none, and the margins of the list elements are adjusted a bit. The image that is a link to the membership form is moved a bit to the left so that it centers better in the navigation menu box. The box is floated to the left when the screen width is larger than 480 pixels.

Content

#content {
    width: 60%;
    float: left;
}

The content section is set to 60% of the page width and floated left to be beside the Main Navigation section when the page width is at least 650 pixels. The content section inherits the black text and the light gray background of the body.

Footer

footer {
	background-color: #eee;
	font-size: .7em;
	text-align: center;
	border-top: #0000D4 solid 5px;
}

footer p {
	margin: 0;
}

The footer is set off from the body using a blue border with a color picked from the logo. The background is set to a darker gray than the body, and the font size is set to 0.7em and aligned center. The margins of the paragraph elements are set to 0 to eliminate spacing between them.
