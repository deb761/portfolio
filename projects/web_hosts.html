What Does a Host do?

A web server is a computer that is connected to the internet (World Wide Web) and serves one or more web sites. To serve a web site, the machine is registered with a Domain Name Server (DNS) and responds to requests to load web pages, transfer files via File Transfer Protocol (FTP), and other services. Without a web server, a company or individual cannot make their content available over the net, even with a domain name. (What Is Web Hosting, 2015)

A web host provides access to a web server to its customers, as well as other associated services, such as domain registration, email, and database support. The customer's web site may be hosted on a shared server, on a dedicated server, or on a virtual (cloud) server.

Shared Server
A shared server hosts multiple sites, and the sites share the resources of that server. (Hosting Reviews, 2015)
Virtual Private Server (VPS)
VPS contains the best elements of shared hosting and dedicated hosting services. Like shared hosting, VPS puts your website on a server that also has other sites running on it, but there are far fewer sites per server than on shared hosting services. But the VPS has its own operating system, RAM, and ports, so it is not affected by other sites hosted on the server. (Wilson, May 4, 2015)
Dedicated Server
A dedicated server hosts just one site, and is needed when a site has heavy traffic. Normally, the customer is responsible for setting up the server and maintaining the software on it. (Wilson, Aug 1, 2015)

What are the Most Important Characteristics for Web Host Service?

A review of several web hosting services revealed a number of different services provided along with just hosting the web site. Here are the most important ones I identified for an individual or small company getting ready to put out their first web site.

24/7 Customer Support
Most companies provide 24/7 customer support, although the responsiveness varies from a few minutes to up to a half hour. Users that have little knowledge about creating a web site will benefit from quicker response times. (Rashid and Wilson, 2015)
Database Support
Many companies provide a database server and the ability to create a number of databases, either MySQL and/or MSSQL, along with tools for defining the database and creating interfaces to the data.
Choice of Host Operating System
Many services offer a choice of a Linux or Windows based operating system, so if you are planning on using any tools specific to one, such an Microsoft SQL Server, you can pick the appropriate OS.
Site Building Tools
These tools provide templats and formatting tools to make creating a website like putting together a PowerPoint presentation.
Email Accounts
The service normally includes email accounts and service using the domain name for your site. Some companies limit the number of addresses that can be defined based on the account level.
Security Features
The service should include security updates, brute force detection, firewalls, and anti-DDoS (Distribited Denial of Service) tools to prevent malicious attacks, and offer SSL (Secure Socket Layer), for secure connections to a site.
Storage Space
The amount of storage space allowed varies by service plan and provider, from 10GB at 1&1 to unlimited at Arvixe.
Traffic Limitations
The companies reviewed didn't meter uploads or bandwidth, but the user agreement on HostGator specified that a shared server user would be in violation of the agreement if the site used more than 25% of system resources for longer than 90 seconds. ("Acceptable Use Policy", 2015)
Statistic Reports
Some sites, such as Arvixe, provide statistic reports on web site usage.

Comparison of Web Hosting Providers

In order to determine which provider would be best, the criteria for selection needs to be defined. For this project, which would be putting together a web site for a small club that needs a site to:  attract new members, showcase upcoming events, and to use a database to track membership and expenses. With this web site use in mind, the requirements are:

  • Shared Server
  • 24/7 Customer Support
  • Database Support - it can be either MySQL or MS SQL
  • Site Building Tools
  • Storage - at least 10GB
  • Backups of data and/or version control
  • Email Accounts - at least 10 included in base price.
  • Security Features to prevent unauthorized access
  • Low cost

Conclusions

Reviewing the various providers, it appears that, aside from free hosting for non-profits, InMotion's Launch is the best choice because it meets all the requirements and has the lowest price barring iPage's introductory price, with iPage's Shared with extras coming in second because of it's low introductory price, even though it has restrictions on server space. The club should go with DreamHost's Shared plan and take advantage of it's free hosting for non-profits.
