What is a Domain Name?

An internet domain name is a string of characters used to identify the location of a certain World Wide Web site. The domain name is registered according to the Domain Name System (DNS) either through an Internet Service Provider (ISP) or through a domain registering service such as the Internet Corporation for Assigned Names and Numbers (ICANN). (Hillstrom and Hillstrom, 2007)

The domain name combined with the protocol specification makes up the Uniform Resource Locator (URL) for a site. An example would be http://www.icann.org/. The domain name for this URL would be "icann.org".

The domain name has two parts: the top-level domain, ".org," and the domain "icann." The "www" in the URL is a subdomain. (Schwabach, 2014)

The DNS correlates the domain name to an actual Internet Protocol (IP) address. The domain name provides an easier and more flexible access to a site, since the domain name can apply to a list of actual server addresses, rather than just one. Because this is a look-up, each domain name must be unique.

In the early days of the internet, cybersquatters registered large numbers of domain names resembling trademarks or other distinctive names with the plan to sell them to the companies that owned that trademark. To resolve this problem, the US Congress enacted the Anticybersquatting Consumer Protection Act (ACPA). This act prvides remedies against known cybersquatter registrants, and also provides a means to transfer the name if the cybersquatter cannot be found. (Schwabach, 2014)

Other disputes over domain names are addressed by the Internet Corporation for Assigned Names and Numbers (ICANN), a nonprofit organization that oversees domain name distribution, as well as the assignation of other identifiers that differentiate one Web site from another. (Gale, 2012)

How Do You Obtain a Domain Name?

There are over 200 approved domain name registrars, some whose primary business is registering domain names, and others are diversified Internet Service companies such as America Online (Public Interest Registry 2013). (Schwabach, 2014) Only a domain name registrar is permitted to access and modify the master database of domain names maintained by InterNIC. Domain names can be registered for a maximum of 10 years. (Registering a Domain Name, 2010)

Many ISPs provide a domain registering service, and will verify that the name is available before registering it for a client. Domain name registration fees vary widely depending on the top-level domain used. For example, a .net domain may cost about $8 a year, while a .blackfriday domain may cost about $36. Potential Domain Names

The ICANN has added a lot of top-level domains over the last few years, and is adding more. That gives a prospective domain name owner a lot of choices. For a personal website, I would think .me, .info, or .net would be good choices. I would choose a site name of perhaps bike-lady.me for a site reflecting mostly my cycling interests. Other choices may include bike-lady.info and bike-lady.net. For a website covering my professional image, it would be deb-engelmeyer.me or .info. All of these domain names are currently available according to all of the above registration sites.

I would choose iPage to register and host the site, partly because I have used the ISP before and know them to be reliable and to have good customer support, and partly because they use carbon-neutral energy for their servers. They also register your domain for free if you use them as a host.
